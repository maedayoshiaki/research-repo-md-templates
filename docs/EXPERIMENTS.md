# EXPERIMENTS.md — 実験ログ

研究の実験を記録するファイル。**再現性**が命。数値の生データは外部（W&B 等）でもよいが、
ここには「仮説・設定・結果の要点・判断」を残す。1 実験 1 ブロック。

## How To Log（記録ルール）

- 実験ごとに ID（`exp-NNN`）を振る。
- **再現に必要な情報**を必ず書く: **commit hash / seed / 設定（config へのリンク or 主要ハイパラ）/ 環境（GPU・主要ライブラリ版）**。
- 結果は主要指標だけ要約。詳細グラフは外部ツールへリンク。
- 結論と次アクションを 1 行で。
